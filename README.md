# Youtube Shorts Autoscroll 🚀

A lightweight browser extension that automatically scrolls to the next YouTube Shorts video when the current one ends. No more manual scrolling!

## ✨ Features
* **Auto-scroll:** Detects when a YouTube Short ends and automatically clicks the "Next" button.
* **No looping:** Disables the default video loop behavior on YouTube Shorts.
* **Lightweight:** Runs silently in the background without affecting browser performance.
* **Cross-Promotion:** Includes a sleek popup UI to discover other cool projects.

## 🛠️ Installation (Developer Mode)
1. Clone or download this repository to your local machine.
2. Open your Chromium-based browser (Chrome, Edge, Brave, etc.) and go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click on **Load unpacked** in the top left corner.
5. Select the folder containing this extension.
6. Open YouTube Shorts, sit back, and enjoy!
